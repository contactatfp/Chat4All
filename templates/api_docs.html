API Documentation

Endpoint: /ask_your_model

This endpoint accepts a POST request with the following parameters in the request body:

  • query: The query that you want to ask your model.
  • persona_template: The name of the persona template to be used. If this is not provided, 'sales_coach_inbound' will be used as the default persona template.

Python example:


    import requests
    import json

    url = "http://localhost:5000/ask_your_model"

    headers = {
        "Content-Type": "application/json"
    }

    data = {
        "query": "What is the meaning of life?",
        "persona_template": "sales_coach_inbound"
    }

    response = requests.post(url, headers=headers, data=json.dumps(data))

    if response.status_code == 200:
        print("Response:", response.json())
    else:
        print("Error:", response.status_code, response.text)
